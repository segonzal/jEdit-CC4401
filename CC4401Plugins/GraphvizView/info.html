Plugin

Plugin que permite visualizar un grafo de Graphviz, a medida que se va escribiendo su código en jEdit.

Archivos

  • actions.xml: Acciones asociadas al plugin.
  • build.xml: Información para compilar plugin.
  • GraphVizView.props: Propiedades asociadas al plugin.
  • info.html: Éste archivo.
  • src/GraphvizView/GraphvizView.java: Funcionalidades del plugin, asociadas a jEdit.
  • src/GraphvizView/GraphvizViewPlugin.java: Núcleo del plugin.
  • src/GraphvizView/Opciones.java: Panel de opciones del plugin.
  • src/GraphvizAPI/GraphvizAPI.java: API de Graphviz.
